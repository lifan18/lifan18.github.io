在hexo用md插入博客图片无法显示

Created2023-02-27|Updated2023-02-27|博客
|Post View:

要点1： 图片命名及路径请用英文字母，多数显示不了的原因是用中文等特殊字符
要点2：将图片文件保存在 Hexo 博客项目的 source 目录中。例如，将图片文件保存在 source/images 目录中。

#其中，图片描述 可以是图片的简短描述，/images/图片文件名 是图片在 source 目录下的路径。例如，如果图片文件名为 example.jpg，则在 Markdown 文件中插入图片的语法
